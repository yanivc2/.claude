אסור להריץ פקודות `gws` — או כל כלי אחר שניגש לחשבון Google של המשתמש — ללא אישור מפורש לפני כל פעולה.

**Why:** המשתמש קבע חוק מפורש אחרי שהפכתי גישה לחשבון ה-Gmail שלו ללא אישור מוקדם.

**How to apply:** לפני כל `gws` command — לעצור ולשאול "מאשר לגשת ל-[שירות] עכשיו?" ולהמתין לאישור. גם אם המשתמש ביקש משימה שדורשת גישה לגוגל — לשאול קודם.
